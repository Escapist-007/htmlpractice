How you'll build...

  1. Set it up
    First, you'll set up the project by planning it out, grabbing the graphics, and setting up the files.
  2. Make it
    Then, you'll put together the actual file with the all of the code that you're writing. You'll fulfill several user stories to make sure that the project is perfect.
  3. Review and push
    Finally, you'll check your project against a list of user stories that need to be fulfilled, before pushing it to Github Pages to show off your work.
