skele-ad.html

This project will have you create an ad for the Skeleton framework.

Step One

For this project, you'll need a simple HTML file titled skele-ad.html. You'll need to link it to a css file tht'll be titled skele-ad.css. You'll also need to download the Skeleton file from getskeleton.com, and link the normalize.css and skeleton.css to your project.

Step Two

View this example site. Done that? Great! Now, here's a list of user stories that you need to fulfill. Don't forget to be creative!

  • I can view functions of Skeleton
  • I can click on a link to view it on Github

Step Three

You're all done. Login to Github, or create an account if you haven't. Then, upload these files into a repository called skele-ad, or something like that. Go under settings, and find Github Pages. You can figure out what to do from here, right?
