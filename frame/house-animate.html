house-animate.html

create an animated house using animate.css

Step One

For this project, you're going to use codepen.io (you can do it on your computer if you want, though). So login/sign up, and create a new pen. Then, in settings, find the CSS tab. Select animate.css for the framework. You can also use other frameworks if you think they might help out other parts of the project.
If you're doing this on your computer, you can create the files of house-animate.html and house-animate.css. You'll also need to download or find a link to the animate.css framework.

Step Two

Visit this example page. Done with that? Great! Here's your requirements:

user stories / requirements

  • At least five different animate.css effects are used
  • It looks like a house
  • It is mostly responsive

Step Three

It's time to be independent. Get this thing on Github. Built this on codepen? Figure out how to download it, then get it up on Github Pages.
