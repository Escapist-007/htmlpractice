animal.html

This project will have you build a page about one of your favorite animals.

Step One

For this project, you'll need a simple HTML file titled animal.html. You'll need to link it to a css file tht'll be titled animal-style.css. If you want to add a favicon --the little image that you see in the tab -- use Google.

Step Two

View this example site. Done that? Great! Now, here's a list of user stories that you need to fulfill. Don't forget to be creative!

  • I can see at least 4 pictures of the animal
  • I can see a list of several facts related to the animal
  • I can see the animal's diet
  • I can view some basic information about the animal
  • I can click on a link to view an external page about the animal
  • The site is responsive

Step Three

You're all done. Login to Github, or create an account if you haven't. Then, upload these files into a repository called animal.html, or something like that. Go under settings, and find Github Pages. You can figure out what to do from here, right?
